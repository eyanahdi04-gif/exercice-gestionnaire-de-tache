mindmap
  root((Application Todo))
    [1. Infrastructure & Tests]
      Connexion Initiale
        console.log Frontend chargé
        fetch ping vers Backend
    [2. Le Coeur : Pattern Observer]
      Observer (Classe Abstraite)
        Rôle : Modèle pour les écouteurs
        Méthode : notify -> Force l'implémentation
      Notifier (Gestionnaire d'alertes)
        Propriété : #observers (Liste privée des abonnés)
        Méthode : addObserver -> Permet de s'abonner
        Méthode : notify -> Alerte tous les abonnés (Bouton général)
    [3. Modèle de Données : Todo]
      Données privées
        #id
        #text
        #done (Converti en vrai ou faux)
      Getters (Accès en lecture)
        id
        text
        done
    [4. Le Cerveau : Controller]
      Hérite de Notifier (Peut envoyer des alertes)
      Stockage : #todos (Tableau d'objets Todo)
      Actions API ( fetch )
        loadTodos (GET) -> Récupère & Convertit en objets
        addTodo (POST) -> Envoie le texte & Ajoute à la liste
        deleteTodo (DELETE) -> Supprime du serveur & Filtre la liste
        toggleTodo (PATCH) -> Inverse l'état & Met à jour la liste
      Fin de chaque action -> Déclenche this.notify()
    [5. L'Visuel : View]
      Hérite d'Observer (Écoute les changements)
      Lien : #controller (Mémorise le controller)
      Démarrage (Constructor)
        S'abonne au Controller
        Écoute le Clic (Bouton Ajouter)
        Écoute la touche Entrée (Input)
      Réception Alerte (notify)
        Relance le rendu de l'écran (#render)
      Rendu Visuel (#render)
        Vide l'ancienne liste HTML
        Boucle sur les todos du Controller
        Crée les éléments (li, checkbox, span, bouton supprimer)
        Branche les clics vers le Controller (toggle, delete)
        Met à jour le compteur
